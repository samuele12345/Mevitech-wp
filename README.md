# Mevitech WordPress Website

This repository contains a WordPress website developed with Elementor for Mevitech Srl, based in Via Bartolomeo Zucchi, Monza, Italy. It includes the theme, essential plugins, and backups to replicate the site in local or staging environments.

## Repository Contents

- `wp-content/` - Custom themes, plugins, and media uploads.
- `backup/` - Full site backup in `.wpress` format (exported via All-in-One WP Migration).
- `.gitignore` - Files and folders excluded from version control.

**Note:** Core WordPress files (`wp-admin` and `wp-includes`) are not included.
